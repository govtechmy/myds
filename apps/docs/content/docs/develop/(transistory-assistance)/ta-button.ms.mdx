## Gambaran Keseluruhan

Butang MYDS HTML/CSS menyediakan variasi yang fleksibel dan gaya yang konsisten untuk aplikasi yang tidak menggunakan React. Ia memastikan pengendalian keadaan yang betul (hover, fokus, dilumpuhkan), serta keharmonian reka bentuk merentasi antara muka sektor awam.

## 1. Persediaan

Ikuti langkah-langkah berikut untuk menyediakan gaya butang dalam projek HTML anda.


### Langkah 1 - Salin Kod CSS

Salin keseluruhan kod CSS yang disediakan dalam tab **CSS** di atas.

Anda boleh:

* **Tambahkan** kod tersebut ke dalam **fail CSS sedia ada** anda (contohnya, `styles.css`, `main.css`, atau fail stylesheet lain yang sudah dihubungkan dalam projek anda),
  **ATAU**
* **Buat** fail baru bernama `styles.css` jika anda belum mempunyai fail stylesheet.

> ⚠ **Penting — Termasuk CSS Reset**
>
> Fail CSS yang disediakan mengandungi **global CSS reset** yang memberi kesan kepada semua elemen HTML (contohnya, membuang margin, padding, border dan gaya lalai pelayar).
> Ini memastikan paparan konsisten merentasi pelayar, tetapi jika projek anda sudah menggunakan reset atau normalize stylesheet sendiri, **sila semak dan laraskan bahagian reset untuk mengelakkan konflik**.

### Langkah 2 - Pautkan CSS dalam HTML

Jika anda telah membuat fail `styles.css` baru, pautkan ia di dalam `<head>` HTML anda:

```html
<link rel="stylesheet" href="styles.css" />
```

Jika anda menambah kod ke fail yang sudah ada, pastikan fail tersebut sudahpun dipautkan dalam HTML anda.


### Langkah 3 - Sertakan font yang diperlukan

Font **Inter** adalah sebahagian daripada panduan reka bentuk MYDS. Tambahkan ia di dalam `<head>` anda:


```html
<link
  href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap"
  rel="stylesheet"
/>
```
`<head>` anda sepatutnya kelihatan seperti berikut:

```html
<head>
  <!-- Inter Font -->
  <link
    href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap"
    rel="stylesheet"
  />

  <!-- Design System Stylesheet -->
  <link rel="stylesheet" href="styles.css" />
</head>
```


## 2. Penggunaan

Semua butang menggunakan kelas asas:

```html
<button class="button-myds">...</button>
```

Untuk menggunakan gaya, tambahkan salah satu kelas pengubah varian berikut:

* `button-myds-primary`
* `button-myds-secondary`
* `button-myds-secondary-color`
* `button-myds-tertiary`
* `button-myds-tertiary-color`
* `button-myds-danger-primary`
* `button-myds-danger-secondary`
* `button-myds-danger-tertiary`

## 4. Keadaan (State)

Setiap varian menyokong:

* **Asal**
* **Arahkan Tetikus (Hover)**
* **Fokus**
* **Dilumpuhkan**


---

## 5. Anatomi

| Pemilih                  | Tujuan                                 |
| ------------------------ | -------------------------------------- |
| `.button-myds`           | Susun atur asas dan tetapan semula     |
| `.button-myds-[variant]` | Menentukan penampilan visual           |
| `:hover`                 | Interaksi apabila tetikus diarahkan    |
| `:focus`                 | Cincin fokus untuk kebolehcapaian      |
| `:disabled`              | Penampilan dilumpuhkan dan gaya kursor |

---

## 6. Penyesuaian

Anda boleh mengatasi gaya lalai dalam fail stylesheet anda
Contoh:

```css
.button-myds.button-myds-primary {
  background-color: #1e40af; /* Warna utama tersuai */
}
```

---

## 7. Kebolehcapaian

* Dibina menggunakan elemen `<button>` asli
* Navigasi papan kekunci disokong dengan cincin fokus yang kelihatan
* Keadaan dilumpuhkan menggunakan atribut `disabled` asli
