## Penggunaan

```ts copy title="Import"
import { ThemeProvider } from "@govtechmy/myds-react/hooks";
import { ThemeSwitch } from "@govtechmy/myds-react/theme-switch";
```

```tsx copy title="Anatomi" title="Pelaksanaan"
export default () => (
  <ThemeProvider>
    <ThemeSwitch />
  </ThemeProvider>
);
```

## Contoh

### Sebagai Suis Togol

Secara lalai, `ThemeSwitch` akan muncul sebagai butang togol, beralih antara mod `terang` dan `gelap`, seperti di bawah.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
  <Tab value="Pratonton">
    <div className="flex justify-center">
      <ThemeSwitch as="toggle" />
    </div>

  </Tab>
  <Tab value="Kod">
```tsx
import { ThemeProvider } from "@govtechmy/myds-react/hooks";
import { ThemeSwitch } from "@govtechmy/myds-react/theme-switch";

<ThemeProvider>
  <ThemeSwitch as="toggle" />
</ThemeProvider>
```

</Tab>
</Tabs>
<Callout
  type="info"
  title="Penting"
  className="bg-bg-primary-50 text-txt-primary border-transparent"
  icon={<InfoIcon className="size-5" />}
>
  Pastikan aplikasi anda dibalut dengan `ThemeProvider` sebelum menggunakan
  komponen `ThemeSwitch`.
</Callout>

### Sebagai Pilihan

Anda juga boleh menggunakan `ThemeSwitch` sebagai menu pilihan dropdown, membolehkan pengguna memilih dari senarai tema.

<Tabs items={["Pratonton", "Kod"]} defaultValue="Pratonton">
<Tab value="Pratonton">
  <div className="flex justify-center">
    <ThemeSwitch as="select" />
  </div>
</Tab>
<Tab value="Kod">
```tsx
import { ThemeProvider } from "@govtechmy/myds-react/hooks";
import { ThemeSwitch } from "@govtechmy/myds-react/theme-switch";

<ThemeProvider>
  <ThemeSwitch as="select" />
</ThemeProvider>
```

  </Tab>
</Tabs>
